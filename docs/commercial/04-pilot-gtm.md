# Pilot GTM — 10 Paid Founding Members

**Objective:** not MRR — **proof**. 10 active clinicians, each writing notes
weekly, with ≥1 documented prevented denial and 3 usable testimonials. That
package is what unlocks the next stage (more pilots / fundraise / EU).

## ICP (who we hunt first)
US **solo & small (1–5) behavioral-health practices that bill insurance**
(Medicaid / BCBS / Aetna / UHC-Optum), write their own notes, and feel denial
pain. Psychotherapists + psychiatric NPs/MDs.
- *Not yet:* pure cash-pay practices (the Denial Shield value is muted) and large
  group practices (longer sales cycle). Phase 2.

## Offer
Founding-member pilot: **free or steeply-discounted for 60–90 days**, then the
standard **$99/mo Solo** (matches `pricing_section.dart`). In exchange: weekly
usage + outcome logging (feeds `03-denial-shield-validation.md`) + a testimonial
if they're happy. Pilot terms: `docs/PILOT-AGREEMENT.md` (+ signed BAA).

## Message (lead with money, not features)
- **Hook:** "How much did insurance deny you last quarter? We catch the denial
  *before* you submit."
- **Proof:** 1 prevented 90837 denial (~$175) > a month of Solo ($99). The tool
  pays for itself on the first catch.
- **Second hook:** "And it writes the note — get ~2 hours/day back."
- Always: decision-support, clinician-in-the-loop, de-identified, you keep
  control of the chart.

## Channels (warm → cold)
1. **Communities:** r/therapists, r/psychotherapy, private-practice FB groups,
   Mental Health Tech / private-practice Slack/Discords. Give value first
   (a free "top 5 denial reasons + fixes" guide built from our rubric).
2. **LinkedIn:** target by title (LCSW/LMFT/LPC/PMHNP) + "private practice."
3. **Associations:** state psychological/counseling association newsletters.
4. **Warm intros:** anyone in our network who is/knows a clinician.
5. **Content/SEO:** "[payer] denial reasons for 90837" — high-intent, ties to the
   product. (See existing `MARKET-RESEARCH-ROADMAP.md`.)

## Funnel
landing waitlist (`landing_waitlist`, live) → 15-min demo → pilot agreement +
BAA → onboarded (BYOK or managed key) → weekly check-in → outcome logging →
testimonial + convert to paid.

## 30 / 60 / 90
- **30:** denial-reasons guide shipped; 50 ICP conversations; 20 demos booked;
  retrospective validation run (`03`, method 3).
- **60:** 10 pilots onboarded + writing notes; outcome logging live.
- **90:** validation memo done; ≥1 documented prevented denial; 3 testimonials;
  ≥3 pilots converted to $99/mo paid. → decide: scale pilots / raise / EU.

## Success metrics
| Metric | Target |
|---|---|
| Active pilots | 10 |
| Notes / pilot / week | ≥ 3 |
| Documented prevented denials | ≥ 1 |
| Testimonials | ≥ 3 |
| Pilot → paid conversion | ≥ 30% |

## Risks
- **No-show usage** — clinicians sign up, don't use it. Mitigate: tight ICP +
  weekly check-ins + the note-writing hook (daily habit, not occasional).
- **Outcome-logging fatigue** — mitigate with the retrospective method first.
- **Legal** — no real PHI before BAA + relay (`01`, `02`).
